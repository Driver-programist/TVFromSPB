- ## 📥 Как добавить плейлист
1. Скопируйте ссылку: https://raw.githubusercontent.com/Driver-programist/TVFromSPB/refs/heads/main/tvfromspb.m3u
2. Скопируйте EPG-источник (Телепрограмма. Показывает что идёт сегодня и будет идти завтра):
https://iptvx.one/EPG,http://epg.one/epg2.xml.gz,https://github.com/matthuisman/i.mjh.nz/raw/master/SamsungTVPlus/us.xml.gz
3. Вставьте в любой IPTV-плеер (VLC, TiviMate, Perfect Player и т.д.)
4. Наслаждайтесь просмотром!
